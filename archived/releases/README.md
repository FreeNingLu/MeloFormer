# 归档版本

本目录包含 MeloFormer 的历史版本，已被 v1.0.1 取代。

## 版本历史

### v1.0.0 (2024-12-02) - 已被 v1.0.1 取代
- 三阶段动态优化初版
- 问题: Phase 2 仍用 num_workers=0，GPU 利用率低

### v0.9.2 (2024-12-02) - 已废弃
- 渐进式 num_workers 切换
- 30 个分片缓存
- 问题: 仍需手动调整 batch_size

### v0.9.1 (2024-12-02) - 已废弃
- 53 个分片缓存（过于激进）
- 问题: 立即 OOM

### v0.9.0 (2024-12-02) - 已废弃
- 动态 num_workers 切换
- Dynamo cache 限制提升
- 问题: Step 10 切换太早

### v0.8.x (2024-12-01) - 已废弃
- FlexAttention + 混合精度支持
- 各种 OOM 修复尝试

## 当前推荐版本

**v1.0.1** - 请使用 MuseFormer 根目录下的最新版本

- 修复 GPU 利用率低问题
- 三阶段优化 (Step 10 切换 num_workers, Step 100 切换 batch_size)
- 基于 H800 80GB 实测数据
